# utf 8 in Windows
append to _vimrc 
```
set encoding=utf-8
set fileencoding=utf-8
set guifont=Lucida_Console:h9:cANSI:qDRAFT
```
# how to understand Lucida_Console:h9:cANSI:qDRAFT
open vim or gvim
```
:set guifont=*
```
select font, Lucida_Console can show utf8
```
set guifont?
```
now you see
```
guifont=Lucida_Console:h9:cANSI:qDRAFT
```
it is all about it
